## [v0.1.0] - 2026-02-06
### Added
- Initial public release of MetalClock for macOS 13+.
- Floating, borderless transparent window with circular hit-testing.
- Menu bar icon with Show/Hide and Quit.
- Metal rendering with SDF and fwidth-based antialiasing.
- Smooth continuous clock hands using high-resolution time.
